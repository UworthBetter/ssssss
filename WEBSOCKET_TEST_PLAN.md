# WebSocket 实时推送功能测试计划

## 测试环境准备

### 前置条件
- [x] 后端服务已启动（端口 8098）
- [x] 前端开发服务器已启动（端口 8080）
- [x] 数据库中有测试数据（服务对象、心率、血压等）
- [x] 用户已登录并具有相应权限
- [x] WebSocket 依赖已添加到 `qkyd-admin/pom.xml`

### WebSocket 端点
- **后端**: `ws://localhost:8098/ws/health/data`
- **协议**: WebSocket
- **支持操作**:
  - 订阅服务对象数据: `{"type": "subscribe", "patientId": xxx}`
  - 取消订阅: `{"type": "unsubscribe"}`
  - 心跳检测: `{"type": "heartbeat"}`
