## 2. 数据流向图

```
异常数据
  │
  ├─→ ai_abnormal_record (原始异常记录)
  │
  ├─→ ai_event_processing_pipeline (管道状态)
  │    ├─ stage: DETECTED
  │    ├─ priority: 95
  │    └─ abnormal_id: xxx
  │
  ├─→ ai_event_insight_snapshot (事件洞察)
  │    ├─ insight_snapshot_id: yyy
  │    └─ 存储到 pipeline.insight_snapshot_id
  │
  ├─→ Python风险评估服务
  │    └─ 返回 risk_score: 85
  │    └─ 存储到 pipeline.risk_score
  │
  ├─→ ai_disposition_rule (规则匹配)
  │    └─ 根据 abnormalType + riskScore 查询规则
  │    └─ 生成 disposition: "立即通知医生"
  │    └─ 存储到 pipeline.disposition_suggestion
  │
  ├─→ ai_operation_audit_log (操作审计)
  │    ├─ DETECT: 异常检测完成
  │    ├─ INSIGHT_BUILD: 事件洞察完成
  │    ├─ RISK_ASSESS: 风险评估完成
  │    ├─ DISPOSITION: 处置建议生成
  │    └─ EXECUTE: 自动执行处置
  │
  └─→ 通知系统 (如果自动执行)
       └─ 发送告警/通知给医生
```
