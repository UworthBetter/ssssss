# Service Manager for 耆康云盾

服务管理控制台 - 一键管理前端、后端、算法服务

## 🚀 快速开始

### Windows 用户
双击运行 `service-manager.bat`，会打开一个图形化菜单：

```
╔════════════════════════════════════════════════════════════╗
║           耆康云盾 - 服务管理控制台                        ║
╚════════════════════════════════════════════════════════════╝

📊 当前端口状态:

  [前端服务] ✅ 运行中 (端口: 8080, PID: 12345)
  [后端服务] ⭕ 未启动 (端口: 8098)
  [Python算法服务] ✅ 运行中 (端口: 5000, PID: 67890)

═══════════════════════════════════════════════════════════

🎯 操作菜单:

  [1] 启动前端服务 (Vue3 - 8080)
  [2] 停止前端服务
  
  [3] 启动后端服务 (Java - 8098)
  [4] 停止后端服务
  
  [5] 启动Python算法服务 (5000)
  [6] 停止Python算法服务
  
  [7] 启动全部服务
  [8] 停止全部服务
  
  [9] 清理所有Node进程 (强制)
  [0] 退出
```

## 📋 功能说明

| 操作 | 说明 |
|------|------|
| 启动前端 | 启动 Vue3 开发服务器，端口 8080 |
| 停止前端 | 终止占用 8080 端口的进程 |
| 启动后端 | 启动 Spring Boot 应用，端口 8098 |
| 停止后端 | 终止占用 8098 端口的进程 |
| 启动算法服务 | 启动 Python 算法服务器，端口 5000 |
| 停止算法服务 | 终止占用 5000 端口的进程 |
| 启动全部 | 按顺序启动所有服务 |
| 停止全部 | 停止所有运行中的服务 |
| 清理Node | 强制终止所有 node.exe 进程 |

## 🔍 实时端口监控

脚本会实时显示以下服务的状态：

| 服务 | 端口 | 进程类型 |
|------|------|----------|
| 前端服务 | 8080 | node.exe |
| 后端服务 | 8098 | java.exe |
| Python算法 | 5000 | python.exe |

## 💡 使用技巧

### 常见场景

1. **启动开发环境**
   ```
   选择 [7] 启动全部服务
   ```

2. **端口冲突解决**
   ```
   选择 [9] 清理所有Node进程
   然后选择 [1] 启动前端服务
   ```

3. **只调试前端**
   ```
   选择 [1] 启动前端服务
   ```

4. **只调试后端**
   ```
   选择 [3] 启动后端服务
   ```

### 手动关闭服务

如果需要手动关闭某个服务的窗口：
- 前端服务：关闭标题为 "前端服务-8080" 的 CMD 窗口
- 后端服务：关闭标题为 "后端服务-8098" 的 CMD 窗口
- Python服务：关闭标题为 "Python算法服务-5000" 的 CMD 窗口

## 🔧 故障排查

### 问题1：端口显示运行中但无法访问
```
解决步骤：
1. 选择停止对应服务
2. 选择 [9] 清理所有Node进程 (如果是前端)
3. 重新启动服务
```

### 问题2：启动后立即关闭
```
可能原因：
- 依赖未安装
- 端口被占用
- 配置错误

解决步骤：
1. 检查对应目录是否存在
2. 检查是否已安装依赖（前端需 npm install）
3. 查看弹出窗口的错误信息
```

### 问题3：多个服务实例同时运行
```
解决步骤：
1. 选择 [8] 停止全部服务
2. 选择 [9] 清理所有Node进程 (强制)
3. 等待 5 秒后重新启动
```

## 📝 注意事项

1. **首次使用**：确保已在各个目录安装依赖
   - 前端：`cd RuoYi-Vue3-Modern && npm install`
   - Python：`cd python-algorithms && pip install -r requirements.txt`

2. **端口占用**：如果 8080/8098/5000 端口被其他程序占用，需要修改配置

3. **权限问题**：某些操作可能需要管理员权限

## 🎯 快捷键

在菜单界面：
- 输入数字选择操作
- 按回车确认
- 按任意键返回主菜单

2. 发送到 → 桌面快捷方式
